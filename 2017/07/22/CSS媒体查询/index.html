CSS媒体查询

2017-07-22

媒体查询是设计响应式网站常用的CSS工具，本文记录下自己在学习CSS媒体查询过程中的心得。

媒体类型

  • all 支持所有媒体类型
  • screen 屏幕
  • print 打印设备
  • braille 盲人设备
  • embossed 盲文打印机
  • handheld 手持小屏设备
  • tv 电视设备，低分辨率

逻辑操作符

  • and 操作符用于合并多个媒体属性或合并媒体类型和媒体属性，相当于js中的&&
  • not 关键字仅能应用于整个查询，而不能单独应用于一个独立的查询。怎么判断独立查询，,号用来分割多个查询，若果有逗号 not作用于最近的一个查询，如果没有逗号， not作用于仅有的一个查询。
  • only 关键字防止老旧的浏览器不支持带媒体属性的查询而应用到给定的样式，

是不支持媒体查询，也就是防止不支持@media的浏览器，直接应用媒体查询后面的css属性。

,号相当于 or操作符
