有个限制name属性只对少数HTML元素有效包括表单 表单元素 <iframe> <img>元素,另外这个方法是定义在HTMLDocument类上。
如果创建一个元素form, <img>, <iframe>, <applet>有name属性，会自动创建document对象对应name属性值为该元素，如果有多个

  • 通过标签名选取元素
  • 通过css类选取元素
  • 通过css选择器

Node对象的属性

  • parentNode 该节点的父节点，Document对象的应该是null.
  • childNodes 只读NodeList对象， 该节点的子节点的实时表示
  • fistChild, lastChild中的第一个和最后一个
  • nextSibling, previoursSibling 该节点的兄弟节点中前一个和下一个。
  • nodeType 节点的类型， 9document 1代表Element, 3代表Text节点，8代表Comment节点， 11代表DocumentFragment节点

这里说一下对Text节点的理解，一个Element节点只要包含文本就包含一个Text子节点，看一下下面这个列子

  • text节点或comment节点的文本内容
  • nodeName 元素标签名

作为元素数(Element对象)的文档

看下面这个列子

我们可以使用忽略元素内容的ElementAPI

  • children 只包含Element对象
  • firstElementChild, lastElementChild,代表第一个Element节点和最后一个Element子节点
  • nextElementSibling previousElementSibling, 前一个，下一个兄弟Element节点
  • childElementCount 子元素的数量

获取元素的内容

  • innerHTML
  • textContent, innerText获取元素的文本内容

增，删

  • 创建节点

创建ELement节点使用document.createElement()给方法传入标签名,/创建Text节点用document.createTextNode，传入字符串文本

复制已存在的节点，Node.cloneNode(node)返回该节点一个全新副本,注意这个方法是在节点实例上调用，不是document

  • 插入节点 appendChild(), insertBefore()

appendChild()是在需要插入节点上调用，如果插入成功这个节点将成为被插入节点的父节点。即被插入节点是最后一个子节点。
insertBefore()传入两个参数，第一个待插入节点，第二个目标节点，待插入节点将在目标节点前面插入，二者是兄弟关系。

  • removeChild() 删除节点和替换节点 replaceChild()

removeChild()在其父节点上调用， replaceChild()在父节点调用，第一个参数新节点， 第二个参数是需要代替的节点
