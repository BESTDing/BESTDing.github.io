JavaScript获取文档尺寸和浏览器滚动

2017-07-23

今天写了个小东西，正好需要js获取元素尺寸位置，而且用到浏览器滚动。记录一下巩固一下今天学到的知识。

文档坐标和视口坐标

  • 视口坐标

是相对于，浏览器窗口元素的坐标，浏览器的窗口取决于显示器的大小和用户的缩放操作。所以元素相对于视口的坐标是不固定的。

  • 文档坐标

文档坐标是元素相对于整个文档的左上角的坐标，这个坐标也是随着浏览器窗口大小改变而改变的。

  • 浏览器滚动

  • 水平滚动

浏览器水平滚动的距离 x 总是大于等于0

  • 垂直滚动

浏览器在垂直方向滚动的距离 y 总是大于等于0

我们可以得出这样的结论

文档坐标 = 视口坐标 + 滚动距离
