ul标签的默认padding

2017-03-19

今天用ul模仿写了一个下面这种效果，无奈自己写的时候发现ul不居中，于是就去看原网站的css代码发现有一下代码

添加之后果然正常了，于是就搜了一下发现
ul标签在FF中默认是有padding值的,而在IE中只有margin默认有值。默认值都是40px.

so~~
ul {
padding-left: 0px;
margin-left: 0px;
}

可以解决一些布局上的问题。
